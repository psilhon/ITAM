# ITAM 系统说明文档

## 1. 系统概述

IT 资产管理系统（ITAM）用于对企业服务器和网络设备进行全生命周期管理，支持资产登记、状态追踪、批量操作、数据导入导出等功能。

| 项目 | 说明 |
|------|------|
| 版本 | v4.5.0 |
| 访问地址 | http://localhost:3001 |
| 技术栈 | React 18 + TypeScript（前端）/ Node.js + Express + Prisma（后端） |
| 数据库 | SQLite（`backend/prisma/dev.db`） |
| 部署方式 | PM2 进程管理 |

## 3. 技术栈详情

### 前端
- React 18 + TypeScript
- Vite 构建工具
- React Router 路由
- TDesign UI 组件库
- Axios HTTP 客户端
- Recharts 图表库
- Framer Motion 动画

### 后端
- Node.js + Express
- Prisma ORM
- SQLite 数据库
- Zod 数据验证
- ExcelJS / xlsx Excel 处理
- bcryptjs 密码加密

## 4. API 接口

### 健康检查
- `GET /api/health` - 服务健康状态

### 版本信息
- `GET /api/version` - 获取系统版本

### 服务器管理
- `GET /api/servers` - 获取服务器列表
- `POST /api/servers` - 创建服务器
- `PUT /api/servers/:id` - 更新服务器
- `DELETE /api/servers/:id` - 删除服务器
- `POST /api/servers/import` - 导入服务器
- `GET /api/servers/export` - 导出服务器

### 网络设备管理
- `GET /api/network-devices` - 获取网络设备列表
- `POST /api/network-devices` - 创建网络设备
- `PUT /api/network-devices/:id` - 更新网络设备
- `DELETE /api/network-devices/:id` - 删除网络设备
- `POST /api/network-devices/import` - 导入网络设备
- `GET /api/network-devices/export` - 导出网络设备

### 仪表盘
- `GET /api/dashboard/stats` - 获取统计数据

## 6. 本地部署

### 6.1 启停命令

```bash
cd /Users/psilhon/WorkBuddy/ITAM

./start-local.sh install  # 安装依赖（首次运行）
./start-local.sh start   # 启动服务
./start-local.sh stop    # 停止服务
./start-local.sh restart # 重启服务
./start-local.sh logs    # 查看日志
./start-local.sh status  # 查看状态
```

### 6.2 PM2 管理

```bash
pm2 list                 # 查看进程
pm2 logs itam-backend    # 查看日志
pm2 restart itam-backend # 重启
pm2 stop itam-backend    # 停止
pm2 save                 # 保存进程列表（开机自启）
pm2 startup              # 配置开机自启
```

### 6.3 数据库

数据库文件位于 `backend/prisma/dev.db`。

如需重置数据库：
```bash
rm backend/prisma/dev.db
cd backend && npx prisma db push && cd ..
./start-local.sh start
```

## 8. 常见问题

**Q: 服务无法启动？**
检查端口是否被占用：`lsof -i:3001`

**Q: 数据库错误？**
重置数据库：
```bash
rm backend/prisma/dev.db
cd backend && npx prisma db push
```

**Q: 前端修改未生效？**
```bash
./start-local.sh restart
```
